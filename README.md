# Telecom Customer Churn Analysis and Prediction

## Overview

This project is a data-driven Telecom Customer Churn Analysis and Prediction System that helps telecom companies identify customers who are likely to leave their services. The system uses machine learning, customer segmentation, association analysis, and forecasting techniques to provide actionable insights.

## Features

* Customer Churn Prediction
* Future Churn Forecasting using Prophet
* Customer Segmentation using K-Means
* Association Rule Mining using Apriori
* REST API Services using FastAPI
* Data Storage using MongoDB
* Interactive Dashboard using React
* Business Intelligence Reports using Power BI

## Workflow

1. Collect telecom subscriber data from data.gov.in API.
2. Perform data preprocessing and feature engineering.
3. Store processed data in MongoDB.
4. Apply machine learning models for churn prediction.
5. Use Prophet for future churn forecasting.
6. Generate insights through React and Power BI dashboards.
